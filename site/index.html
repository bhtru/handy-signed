macOS 13+ menu-bar utility

Handy

Finder shortcuts and small system controls for people who miss the fast parts of Windows.

Download for macOS View source

Developer ID signed releases require macOS Ventura or later.

What it adds

Small Finder changes that remove daily friction.

01

Cut and paste files

Use Command-X and Command-V in Finder to move files with undo support.

02

Return opens files

Press Return to open selected items while Shift-Return keeps rename close by.

03

Context menu helpers

Copy Path, Open in Terminal, New .txt File Here, and Extract All Here live where you right-click.

04

System toggles

Keep Awake, hidden files, dark mode, Bluetooth sleep behavior, and Empty Trash sit in one quiet utility.

Install once

Drag to Applications, then grant only the permissions you use.

  1. 1Download the DMG and move Handy into Applications.
  2. 2Enable the Finder Extension for right-click tools.
  3. 3Grant Accessibility and Finder Automation for keyboard shortcuts.
