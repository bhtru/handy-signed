## What is included

Handy 1.0 is centered on Finder workflows that longtime Windows users often expect by default:

- Cut and paste files with Command-X and Command-V.
- Open selected files with Return.
- Copy a file path from the Finder context menu.
- Open folders in Terminal, preferring iTerm2 when it is installed.
- Create a new `.txt` file from empty Finder space.
- Extract ZIP archives without leaving `__MACOSX` folders behind.

## Permission model

The app only asks for permissions tied to the features you enable. Finder Extension powers right-click actions, Accessibility powers keyboard shortcuts, and Finder Automation allows Handy to inspect and move the selected files.

## Distribution

The release pipeline produces a drag-to-Applications DMG, signs the app with Developer ID, and supports Apple notarization when the notary profile is available.
